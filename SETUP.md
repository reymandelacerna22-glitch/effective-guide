# SETUP.md

## Hardware Requirements
- FPGA Development Board: Specify the model and specifications needed for your project.
- Power Supply: Ensure you have the appropriate power supply for your FPGA board.

## Toolchain Setup Instructions

### Quartus
1. Download and install Quartus from the Intel website.
2. Open Quartus and set up a new project.
3. Configure the device family according to your FPGA model.

### Vivado
1. Download Vivado from the Xilinx website.
2. Install Vivado following the on-screen instructions.
3. Create a new project and select the appropriate FPGA device.

### iCEcube2
1. Download iCEcube2 from the Lattice Semiconductor website.
2. Install the software and configure the settings based on your FPGA.
3. Start a new project and select the correct device.

## Step-by-Step FPGA Configuration Guide
1. Connect your FPGA development board to your computer.
2. Open the toolchain you are using (Quartus, Vivado, or iCEcube2).
3. Load your design files into the toolchain.
4. Compile the design to generate the programming file.
5. Use the appropriate programming tool to upload the design to the FPGA.
6. Verify that the FPGA is functioning as expected.
